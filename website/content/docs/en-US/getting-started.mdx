# Getting started

This guide walks through building the `chic` compiler, creating a small project, and running your first Chic program.

## Prerequisites

- Rust (stable toolchain)
- `clang` (required for some native/LLVM workflows; CI installs it on Linux and `brew install llvm` on macOS)

## Build the compiler

```sh
cargo build --bin chic
./target/debug/chic --help
```

## Create a new project

```sh
./target/debug/chic init --template app --output ./hello
cd ./hello
```

The generated project includes:

- `manifest.yaml` — project metadata and build settings.
- `src/` — Chic source files (`.ch`).
- `tests/` — Chic `testcase` files.
- `docs/` — project-specific documentation.

## Write your first program

Open `src/App.ch` and change `Main` to something simple:

```chic
namespace Hello;

public static class Program
{
    public static int Main(string[] args)
    {
        Std.Console.WriteLine("Hello, Chic!");
        return 0;
    }
}
```

## Build and run

```sh
./target/debug/chic build .
./target/debug/chic run .
```

## Run tests

```sh
./target/debug/chic test .
```

See also: `docs/testing/README.md`.
